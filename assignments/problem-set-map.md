# CS229 Problem Set and Exercise Map

## 1. Purpose

This file maps CS229 lectures to assignment gates. The goal is to prevent passive lecture learning. Each lecture cluster should be followed by derivation practice, implementation practice, and public-safe reflection.

## 2. Academic Integrity Boundary

Allowed:

* high-level mapping
* personal derivation summaries
* original implementations
* experiment reports
* mistake logs
* public-safe reflections

Not allowed:

* official solution keys
* copied student solutions
* full restricted assignment prompts
* answer dumps
