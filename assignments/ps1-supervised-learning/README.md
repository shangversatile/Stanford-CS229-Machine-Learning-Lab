# PS1 Supervised Learning Gate

## 1. Scope

This gate covers:

* Lecture 2: linear regression, gradient descent, normal equation
* Lecture 3: LWR, probabilistic interpretation, logistic regression, perceptron
* Lecture 4: Newton method, exponential family, GLM, softmax

## 2. Why This Gate Matters

PS1 is the first real test of CS229 supervised learning. It forces the learner to move from lecture familiarity to derivation accuracy, optimization implementation, and error analysis.
