# Academic Integrity Statement

This repository is for independent learning and research preparation.

Stanford CS229 official assignment solutions must not be published in this repository.

Public student solutions must not be copied into this repository.

GitHub content in this repository should be limited to independent implementations, personal derivations, experiment reports, reflections, and non-infringing materials.

If a public implementation is consulted, the source must be cited clearly, and the reference may only be used for debugging or understanding an idea after an independent attempt.

Official assignment files, solution keys, private course materials, and copyrighted materials must not be committed to the public repository.
